# 📊 Power BI Sales Dashboard

## 📖 Project Overview

This project is an interactive Sales Dashboard created using Microsoft Power BI. It provides insights into revenue, sales performance, category-wise analysis, and business trends.

## 📌 Key Insights

* Total Revenue Analysis
* Net Revenue Tracking
* Category-wise Performance
* Revenue Trends
* Product Analysis

## 🛠 Tools Used

* Business Intelligence
